So, if you'll notice this blog's homepage utilizes iframes. I had to figure out (with much aid from our friendly netizens) how to make the iframe have the same height as the content.

不知道你們有沒有主義到，這網站是利用iframe的。由網友們的幫助才發現如何把iframe高度調整和內容高度同樣。

var iframeClass = document.getElementsByTagName("iframe");
sizeTheFrames();
function sizeTheFrames() {
	for (var i = 0; i < iframeClass.length; i++) {
		var x = document.getElementsByTagName("iframe").item(0);
		x.onload = function () {
			x.style.height = x.contentWindow.document.body.scrollHeight + 'px';
		}
	}
}
